# Deepfake 检测平台前端

一个简洁、美观、易用的 Deepfake 检测平台前端应用，与后端 API 完美集成，提供直观的用户界面进行深度伪造检测。

## 🚀 功能特性

### 核心功能
- **🔍 智能检测** - 支持图片和视频文件的深度伪造检测
- **📁 拖拽上传** - 简单直观的文件上传体验
- **🧠 多模型支持** - 支持 VGG、LRCN、Swin Transformer、ViT、ResNet 等多种检测模型
- **📊 实时结果** - 即时显示检测结果和置信度
- **📈 数据统计** - 详细的检测统计和趋势分析

### 管理功能
- **📜 历史记录** - 完整的检测历史查看和管理
- **🔧 模型管理** - 查看和管理可用的检测模型
- **📉 统计分析** - 可视化的检测数据和性能指标
- **🗑️ 记录管理** - 支持删除历史检测记录

### 用户体验
- **📱 响应式设计** - 完美适配桌面和移动设备
- **🎨 现代界面** - 简洁美观的用户界面设计
- **⚡ 快速响应** - 优化的性能和流畅的交互
- **🔔 智能提示** - 友好的操作反馈和错误提示

## 🛠️ 技术栈

- **HTML5** - 语义化标记和现代 Web 标准
- **CSS3** - 响应式设计和动画效果
- **JavaScript (ES6+)** - 现代 JavaScript 特性
- **Axios** - HTTP 请求库
- **Chart.js** - 数据可视化
- **Font Awesome** - 图标库

## 📋 系统要求

- 现代浏览器（Chrome 80+、Firefox 75+、Safari 13+、Edge 80+）
- 后端 API 服务运行在 `http://localhost:8000`
- 网络连接（用于加载 CDN 资源）

## 🔧 配置说明

### API 配置
在 `script.js` 中修改 API 基础 URL：
```javascript
const API_BASE_URL = 'http://localhost:8000/api/v1';
```

### 支持的文件格式
- **图片**: JPG, JPEG, PNG, GIF, BMP, WEBP
- **视频**: MP4, AVI, MOV, MKV, WMV

### 文件大小限制
- 单个文件最大 100MB
- 支持批量上传（最多 100 个文件）

## 📖 使用指南

### 1. 文件检测
1. 在检测页面，点击上传区域或拖拽文件
2. 选择检测模型（默认 VGG）
3. 点击"开始检测"按钮
4. 等待检测完成，查看结果

### 2. 查看历史
1. 点击导航栏的"历史"
2. 使用筛选器过滤记录
3. 点击删除按钮移除不需要的记录

### 3. 模型管理
1. 点击导航栏的"模型"
2. 查看可用模型的状态和统计
3. 了解各模型的性能指标

### 4. 统计分析
1. 点击导航栏的"统计"
2. 查看总体检测数据
3. 分析检测趋势图表

## 🔍 API 集成

### 检测接口
```javascript
POST /api/v1/detection/detect
Content-Type: multipart/form-data

参数:
- file: 上传的文件
- model_type: 模型类型
```

### 历史记录接口
```javascript
GET /api/v1/detection/history
参数:
- skip: 跳过记录数
- limit: 限制记录数
- prediction: 结果筛选
- model_type: 模型筛选
```

### 统计接口
```javascript
GET /api/v1/detection/statistics
返回: 检测统计数据
```

## 🐛 故障排除

### 常见问题

**Q: 无法连接到后端 API**
A: 检查后端服务是否在 `http://localhost:8000` 运行

**Q: 文件上传失败**
A: 确认文件格式和大小符合要求

**Q: 检测结果不显示**
A: 检查浏览器控制台的错误信息

**Q: 页面样式异常**
A: 清除浏览器缓存或检查网络连接

### 调试模式
打开浏览器开发者工具查看：
- Console 面板：JavaScript 错误和日志
- Network 面板：API 请求状态
- Elements 面板：DOM 结构和样式
